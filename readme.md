# About The Project 

This project is a front-end clone of the official *Myntra* website.

The goal of this project was to strengthen my HTML and CSS skills by recreating a real-world e-commerce website.

Instead of simply copying the design, I built every section from scratch and solved different UI challenges related to layouts, spacing, image positioning, typography, Flexbox, and responsive structure.

# Challenges Faced

Building this project wasn't easy.

Same of the challenges I solved were:

- Creating the promotional banner
- Matching the original Myntra spacing
- Creating equal-sized category cards
- Proper image alignment
- Adding realistic category names and discounts 
- Organizing HTML and CSS efficiently
- Making the design closely resemble the original website

Every challenge improved my problem-solving skills.
